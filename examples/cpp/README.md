# Examples

This folder contains examples of LlamaLib usage in C++:
- [basic](basic): The basic example demonstrates how to start a LLM and run its main methods
- [basic_embeddings](basic_embeddings): The basic_embeddings example demonstrates how to generate vector embeddings for text.
- [agent](agent): The agent example demonstrates the agent functionality.
- [remote_client](remote_client): The remote client example demonstrates the remote server-client functionality.
