# Examples

This folder contains examples of LlamaLib usage in C#:
- [basic](basic): The basic example demonstrates the `LLMService` class
- [local_client](local_client): The local client example demonstrates the local client functionality of the `LLMClient` class. 
- [remote_client](remote_client): The remote client example demonstrates the remote server-client functionality of the `LLMClient` class.
- [agent](agent): The agent example demonstrates the agent functionality of the `LLMAgent` class.
